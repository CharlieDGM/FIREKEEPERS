# PCB

Esta carpeta contiene los archivos del PCB que realizamos para el proyecto.
Se encuentra el esquematico de todos los componentes utilizados en el proyecto y el zip GERBER con todos los archivos necesarios en caso de realizar la impresion con maquinas especializadas.

La placa de circuitos esta dise√±ada para ser utilizada por un controlador ESP32. A un costado del PCB se encuentran todas las conexiones para los controladores de velocidad ESC 30A para motores brushless. En otro costado se encuentran todas las conexiones para poder conectar los sensores de temperatura necesarios para el funcionamiento correcto del proyecto. A un lado se encuentran los diferentes pines para conectar el ESP32 con una RaspberryPI.
